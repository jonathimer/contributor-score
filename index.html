Contributor Score

A reputation score for open source contributors

What is Contributor Score?

A score from 0-100 based on public GitHub data, considering account age, PR acceptance rate, contribution history, and more. Think "credit score for open source".

Why do we need this?

Open source is at a turning point. Many maintainers stopped accepting outside contributions because of low-quality, AI-generated pull requests. This score could help maintainers systematically filter out low-quality/spam contributions.
